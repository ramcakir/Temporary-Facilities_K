# Temporary Facilities Cost Estimator

A Progressive Web App (PWA) for estimating temporary facilities costs for large construction projects. Designed for Dutch and Western European pricing levels.

## Features

- **Dual Pricing Modes**: Toggle between Rent/Lease and Purchase options
- **Comprehensive Categories**:
  - Mobilisation & Demobilisation
  - Temporary Buildings (containers, tents)
  - Offices, Meeting Rooms & HVAC
  - Welfare Facilities (toilets, kitchens, canteens)
  - Services (cleaning, security, maintenance)
  - Utilities (power, water, internet)

- **Editable Unit Rates**: Customize all rates to match current market prices
- **Project Duration Calculations**: Automatic monthly cost projections
- **Data Persistence**: Save and load projects as JSON files
- **Export Options**: PDF (via print) and Excel/CSV export
- **Offline Support**: Works without internet connection once installed

## Pricing Reference (Dutch/Western Europe 2024)

### Rental Rates (per month)
- Modular office units: €300-400/unit
- Storage containers: €100-150/unit
- Portable toilets: €75-100/unit
- Security services: €4,000-5,000/month per guard
- Cleaning services: €800-1,000/month

### Purchase Rates
- Modular office units: €8,000-12,000/unit
- Storage containers: €2,000-3,000/unit
- Site mobilisation: €10,000-20,000 (one-time)

## Installation

1. Save all files to a web server or local directory
2. Ensure icons are placed in an `icons/` folder
3. Access via HTTPS (required for PWA features)
4. Add to home screen when prompted
